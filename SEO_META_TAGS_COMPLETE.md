## üéØ SEO Strategy

### 1. **Title Tag Structure**

Format: `Primary Keyword - Value Proposition | Qualifier`

**Examples**:
- Homepage: `Belgische Chocolade Gids - Lokale & Internationale Gids | 470+ Winkels`
- Directory: `Directory - 470+ Chocolatiers op Kaart | Zoek per Provincie`

**Benefits**:
- Clear value proposition
- Keywords at the beginning
- Pipe separator for readability
- Under 60 characters where possible

### 2. **Description Tag Strategy**

**Length**: 150-160 characters
**Structure**: 
1. Opening hook (what we offer)
2. Key features/brands
3. Call to action or benefit

**Examples**:
- Homepage: Mentions specific brands (Godiva, Neuhaus) + features (search, map)
- Directory: Highlights interactivity and filtering options

### 3. **Keywords Strategy**

**Focus Areas**:
- **Dutch keywords**: Primary audience is Belgian/Dutch-speaking
- **Brand names**: Godiva, Neuhaus, Leonidas, Pierre Marcolini, Wittamer
- **Location terms**: Belgi√´, Belgische, per provincie
- **Product terms**: chocolatiers, chocolade winkels, ambachtelijke chocolade
- **English terms**: Belgian chocolate guide (for international audience)

### 4. **Social Media Optimization**

**Open Graph Tags** (Facebook, LinkedIn):
- `og:title` - Shorter, punchier version of title
- `og:description` - Concise value proposition
- `og:type` - website or article
- `og:url` - Canonical URL

**Twitter Cards**:
- `twitter:card` - summary_large_image
- `twitter:title` - Same as OG title
- `twitter:description` - Same as OG description
